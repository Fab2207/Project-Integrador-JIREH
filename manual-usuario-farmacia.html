1. Introducción

El Sistema de Gestión Farmacia JIREH es una aplicación integral diseñada para administrar todas las operaciones de una farmacia, desde la gestión de inventario hasta las ventas y reportes financieros.

Características Principales

Gestión de Inventario

Control completo de productos, categorías y stock en tiempo real.

Ventas y Facturación

Procesamiento de ventas con generación automática de boletas.

Gestión de Compras

Control de compras a proveedores con actualización automática de stock.

Reportes Detallados

Informes completos de ventas, compras y movimientos de inventario.

Requisitos del Sistema

  • Java Runtime Environment (JRE) 8 o superior
  • MySQL Server 5.7 o superior
  • Conexión a base de datos configurada
  • Impresora (opcional, para impresión de documentos)

2. Acceso al Sistema

Inicio de Sesión

1 Ejecute la aplicación desde el archivo JAR o acceso directo.
2 Se mostrará la pantalla de login con los campos Usuario y Contraseña.
3 Ingrese sus credenciales proporcionadas por el administrador.
4 Haga clic en el botón "Login" para acceder al sistema.

⚠️ Importante

Si olvida su contraseña, contacte al administrador del sistema. Solo los administradores pueden restablecer contraseñas.

Credenciales por Defecto

El sistema viene con usuarios predeterminados para pruebas:

  • Administrador: Usuario: admin / Contraseña: admin123
  • Auxiliar: Usuario: auxiliar / Contraseña: aux123

3. Roles y Permisos

Administrador ADMIN

Tiene acceso completo a todas las funcionalidades del sistema:

  • ✅ Gestión completa de productos (crear, editar, eliminar)
  • ✅ Gestión de compras y proveedores
  • ✅ Procesamiento de ventas
  • ✅ Gestión de empleados y clientes
  • ✅ Gestión de categorías
  • ✅ Acceso a todos los reportes
  • ✅ Configuración del sistema
  • ✅ Impresión de documentos

Auxiliar/Recepcionista AUXILIAR

Acceso limitado a funciones operativas diarias:

  • ✅ Visualización de productos (solo lectura)
  • ✅ Procesamiento de ventas
  • ✅ Gestión de clientes
  • ❌ No puede crear/editar/eliminar productos
  • ❌ No puede gestionar empleados
  • ❌ No puede gestionar proveedores
  • ❌ No puede gestionar categorías
  • ❌ Acceso limitado a reportes
  • ❌ No puede acceder a configuración

💡 Nota

Los permisos se verifican automáticamente. Si intenta acceder a una función restringida, el sistema mostrará un mensaje de "No tienes privilegios de administrador".

5. Gestión de Productos ADMIN

Registrar Nuevo Producto

1 Haga clic en la pestaña "Productos" en el menú lateral.
2 Complete los campos obligatorios:
  • Código: Código único del producto
  • Nombre: Nombre del medicamento/producto
  • Descripción: Descripción detallada
  • Precio: Precio unitario de venta
  • Categoría: Seleccione de la lista desplegable
3 Haga clic en "Registrar" para guardar el producto.

Editar Producto Existente

1 En la tabla de productos, haga clic en la fila del producto a editar.
2 Los datos se cargarán automáticamente en el formulario.
3 Modifique los campos necesarios.
4 Haga clic en "Actualizar" para guardar los cambios.

Eliminar Producto

1 Seleccione el producto en la tabla.
2 Haga clic en "Eliminar".
3 Confirme la eliminación en el diálogo que aparece.

⚠️ Importante

No se pueden eliminar productos que tengan movimientos de inventario asociados (ventas o compras). El sistema reorganizará automáticamente los IDs después de una eliminación.

Búsqueda de Productos

Utilice el campo de búsqueda para filtrar productos por nombre. La búsqueda es en tiempo real y no distingue entre mayúsculas y minúsculas.

6. Gestión de Compras ADMIN

Realizar Nueva Compra

1 Acceda a la pestaña "Compras".
2 Seleccione el proveedor del menú desplegable.
3 Agregue productos a la compra:
  • Ingrese el código del producto y presione Enter
  • Se cargarán automáticamente el nombre y precio
  • Ingrese la cantidad a comprar
  • Modifique el precio si es necesario
  • Haga clic en "Agregar Producto"
4 Repita el paso 3 para todos los productos de la compra.
5 Verifique el total calculado automáticamente.
6 Haga clic en "Confirmar Compra" para finalizar.

💡 Funcionalidades Automáticas

  • El stock se actualiza automáticamente al confirmar la compra
  • Se genera una factura de compra que se puede imprimir
  • No se pueden mezclar proveedores en una misma compra
  • El subtotal se calcula automáticamente (cantidad × precio)

Eliminar Productos de la Compra

Antes de confirmar la compra, puede eliminar productos seleccionándolos en la tabla y haciendo clic en "Eliminar".

Nueva Compra

Use el botón "Nueva Compra" para limpiar todos los campos y comenzar una nueva transacción.

7. Gestión de Ventas AMBOS

Procesar Nueva Venta

1 Acceda a la pestaña "Ventas".
2 Registre el cliente:
  • Ingrese el DNI del cliente y presione Enter
  • Si existe, se cargará automáticamente el nombre
  • Si no existe, deberá registrarlo primero en la sección Clientes
3 Agregue productos a la venta:
  • Ingrese el código del producto y presione Enter
  • Se mostrarán nombre, precio y stock disponible
  • Ingrese la cantidad deseada
  • Verifique que hay stock suficiente
  • Haga clic en "Agregar Producto"
4 Repita el paso 3 para todos los productos de la venta.
5 Verifique el total de la venta.
6 Haga clic en "Confirmar Venta" para finalizar.

💡 Funcionalidades Automáticas

  • Verificación automática de stock disponible
  • Actualización automática del inventario
  • Generación automática de boleta de venta
  • Cálculo automático de subtotales y total
  • Prevención de venta de productos sin stock

Gestión Durante la Venta

  • Eliminar productos: Seleccione el producto en la tabla y haga clic en "Eliminar"
  • Nueva venta: Use "Nueva Venta" para limpiar y comenzar una nueva transacción
  • Verificar stock: El sistema muestra el stock disponible al agregar cada producto

⚠️ Importante

Una vez confirmada la venta, no se puede modificar. Asegúrese de verificar todos los datos antes de confirmar. El stock se reduce automáticamente y se genera la boleta.

Muestra todas las ventas realizadas con la siguiente información:

  • Número de boleta
  • Cliente
  • Empleado que realizó la venta
  • Total de la venta
  • Fecha y hora

Reporte de Compras

Muestra todas las compras realizadas con:

  • Número de factura
  • Proveedor
  • Total de la compra
  • Fecha y hora

Acceso a Reportes

1 Haga clic en la pestaña "Reportes".
2 Los reportes se cargan automáticamente.
3 Use las pestañas para alternar entre reportes de ventas y compras.

💡 Funcionalidades

  • Los reportes se actualizan automáticamente
  • Ordenamiento por fecha (más recientes primero)
  • Información detallada de cada transacción

14. Impresión de Documentos

Boleta de Venta

Se genera automáticamente al confirmar una venta:

  • Se abre una ventana con la boleta formateada
  • Incluye datos del cliente, productos vendidos y total
  • Haga clic en "Imprimir" para enviar a la impresora

Factura de Compra

Se genera automáticamente al confirmar una compra:

  • Muestra detalles de la compra realizada
  • Incluye proveedor, productos y empleado que realizó la compra
  • Haga clic en "Imprimir" para imprimir

Configuración de Impresora

1 Asegúrese de tener una impresora configurada en el sistema.
2 Al hacer clic en "Imprimir", se abrirá el diálogo de impresión del sistema.
3 Seleccione la impresora y configure las opciones.
4 Confirme la impresión.

💡 Consejos

  • Use papel tamaño A4 para mejor presentación
  • Verifique que la impresora tenga tinta/tóner suficiente
  • Los documentos se formatean automáticamente para impresión

15. Solución de Problemas

Problemas Comunes

🔧 Error de Conexión a Base de Datos

Síntoma: El sistema no inicia o muestra errores de conexión.

Solución:

  • Verifique que MySQL Server esté ejecutándose
  • Confirme las credenciales de conexión (usuario: root, contraseña: admin)
  • Asegúrese de que la base de datos "farmacia_database" exista
  • Verifique que el puerto 3306 esté disponible

🔧 No Puedo Acceder a Ciertas Funciones

Síntoma: Mensaje "No tienes privilegios de administrador".

Solución:

  • Verifique su rol de usuario (Administrador vs Auxiliar)
  • Solo administradores pueden acceder a gestión de empleados, proveedores, etc.
  • Contacte al administrador para cambio de permisos

🔧 Error al Imprimir

Síntoma: Los documentos no se imprimen correctamente.

Solución:

  • Verifique que la impresora esté conectada y encendida
  • Confirme que los drivers estén instalados
  • Pruebe imprimir desde otra aplicación
  • Reinicie el servicio de impresión del sistema

🔧 Productos Duplicados o Errores de Stock

Síntoma: Stock incorrecto o productos duplicados.

Solución:

  • Verifique que no haya códigos de producto duplicados
  • Revise las transacciones de compras y ventas
  • Contacte al administrador para revisión de base de datos

  • proveedores: Información de proveedores
  • ventas/ventas_detalles: Transacciones de venta
